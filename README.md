# Snake Game

Terminal-based Snake game

![scrrenshot](http://i.imgur.com/pHf4fjt.gif)

## Play

### Locally

```
$ go get github.com/techieIsaac/snake-game
$ $GOPATH/bin/snake-game
```

### On Docker

```
$ docker run -ti snake-game
```

## Testing

```
$ cd $GOPATH/src/github.com/techieIsaac/go-snake-game
$ make
```
